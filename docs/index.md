# Tutor Central

[![CI Status](https://github.com/AY2526S2-CS2103T-T09-2/tp/workflows/Java%20CI/badge.svg)](https://github.com/AY2526S2-CS2103T-T09-2/tp/actions)
[![codecov](https://codecov.io/gh/AY2526S2-CS2103T-T09-2/tp/graph/badge.svg?token=BF1CJ5ACX9)](https://codecov.io/gh/AY2526S2-CS2103T-T09-2/tp)

![Ui](images/Ui.png)

**Tutor Central is a desktop application for freelance tuition teachers to manage their students.**

It lets you track student details, lesson schedules, and payment statuses — all from a fast, keyboard-driven interface. While it has a GUI, most interactions happen via a CLI-style command box.

* If you're interested in using Tutor Central, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you're interested in contributing to Tutor Central, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
